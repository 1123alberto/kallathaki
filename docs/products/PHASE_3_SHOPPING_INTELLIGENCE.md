# Kallathaki Phase 3
# Shopping Intelligence

Version: 1.0

---

# Mission

Kallathaki evolves from a shopping optimizer into a personal grocery assistant.

Instead of reacting to searches...

It begins anticipating needs.

---

# Vision

The app should know:

what users usually buy

when they buy it

where they buy it

how much they normally spend

when prices become attractive

---

# Core Principle

Don't wait for the user to ask.

Help first.
